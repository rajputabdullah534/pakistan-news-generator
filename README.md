# Pakistan Breaking News Generator 🇵🇰

Funny Python project — generates random satirical Pakistani headlines.

## Features
- 4 categories: Drama, Cricket, Random, Custom
- Enter your own person & place (Custom mode)
- Auto-save headlines to .txt file
- Session tracker with timestamps

## Run
```
python breaking_news.py
```
